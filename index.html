Gritano

The easiest way to set up a git server

Create repositories and manage user access using this practical tool

Get Started!

Easy to install

Gritano was built to be very easy to install. All you need to do is execute few commands.

Private

Set up your own git server and choose who will be able to access your projects.

SSH

Use SSH public keys to control who has access to your git server.

Terminal

Do you want to create a new repository? Add a new user? All you need is your terminal. Just that.

Remote Control

You can do some tasks remotely like adding ssh pubkeys or listing your repositories.

Open Source

Gritano is a open source project writen in ruby. View the source code on Github.
